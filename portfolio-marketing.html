Dashboard Excel  ·  Social Analytics  ·  A/B Testing

Engoo (DMM.com Group) —
Dashboard de performance Xiaohongshu

Situation · Contexte & Défi

Engoo (groupe DMM.com) cible les apprenants sinophones via Xiaohongshu (小红书) — réseau de 300 M+ utilisateurs actifs, mi-Instagram mi-Pinterest. À mon arrivée : présence sporadique, aucun outil de suivi structuré, décisions basées sur l'intuition. Le défi : mesurer objectivement quel contenu génère de l'engagement, sans budget analytics tiers — XHS n'expose pas d'API publique.

Action · Construction du dashboard

Conception d'un classeur Excel à 3 feuilles entièrement autonome :

  • Données brutes : 12 métriques quotidiennes — vues, likes, sauvegardes, partages, commentaires, abonnés, taux d'engagement, CTR, taux de complétion vidéo, portée.
  • Dashboard dynamique (TCD) : courbes temporelles par KPI, graphiques de comparaison des formats, moyennes mobiles. Zéro mise à jour manuelle.
  • Module A/B dynamique : deux listes déroulantes pour sélectionner n'importe quelles deux périodes — calcul automatique des écarts en % pour chaque KPI.
Mise en forme conditionnelle pour les outliers, formules DÉCALER+INDIRECT pour fenêtres glissantes.

Résultats · Impact sur 3 mois

+93.3% de vues · +89.1% d'interactions · +69.7% d'abonnés — mesurés et documentés via le dashboard.
Le contenu éducatif surperforme le promotionnel (2–3× plus de sauvegardes). Le module A/B a validé +34% CTR et +28% engagement après pivot vers un format tutoriel court + visuel épuré. Le dashboard est devenu l'outil de décision hebdomadaire de l'équipe.
