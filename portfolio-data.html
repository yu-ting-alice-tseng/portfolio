Dashboard Excel – Performance Xiaohongshu

Construction d'un tableau de bord Excel complet pour suivre 12 métriques quotidiennes du compte Xiaohongshu d'Engoo — KPIs, évolution temporelle et comparaison A/B de périodes.

Situation · Problème analytique

Aucun outil de suivi structuré pour le compte Xiaohongshu d'Engoo (groupe DMM.com). Décisions de contenu basées sur l'intuition. Objectif : mesurer objectivement l'impact des décisions éditoriales sur 12 KPIs, sans outil analytics tiers (XHS n'expose pas d'API publique).

Action · Architecture du dashboard

Classeur Excel à 3 feuilles :

  • Données brutes : saisie quotidienne de 12 métriques (vues, likes, sauvegardes, partages, commentaires, abonnés, taux d'engagement, CTR, taux de complétion vidéo, portée). Mise en forme conditionnelle pour les jours outliers.
  • Dashboard dynamique (TCD) : courbes temporelles par KPI avec fenêtres glissantes (formules DÉCALER+INDIRECT), graphiques de comparaison des formats de contenu. Zéro mise à jour manuelle.
  • Module A/B dynamique : deux listes déroulantes pour sélectionner n'importe quelles deux périodes — calcul automatique des écarts en % pour chaque KPI.

Résultats · Impact mesuré sur 3 mois

+93.3% vues · +89.1% interactions · +69.7% abonnés — documentés via le dashboard. Module A/B : +34% CTR et +28% engagement après pivot vers tutoriel court + visuel épuré. Le dashboard est devenu l'outil de décision hebdomadaire de l'équipe.

Risky Deals : M&A transfrontalières & Risque Politique en Amérique Latine

Étude empirique (2000–2024) · 3 hypothèses · Données : LSEG Workspace / Refinitiv, WGI, POLCON III, FSI

Situation

L'Amérique Latine présente un paradoxe pour les acquéreurs : ses ressources naturelles et son potentiel de croissance attirent les capitaux étrangers, mais son instabilité politique chronique génère un risque difficilement quantifiable. Comment l'instabilité politique influence-t-elle concrètement les décisions M&A cross-border ?

Action — Méthodologie

  • H1 – Volume : Régression OLS sur panel data — l'indice WGI prédit-il le nombre d'opérations M&A annuelles ?
  • H2 – Abandon : Modèle Logit — la dégradation du WGI augmente-t-elle la probabilité d'annulation de transaction ?
  • H3 – Prime : Régression OLS — les acheteurs paient-ils une prime plus élevée pour compenser le risque politique ?
  • Sources : LSEG Workspace / Refinitiv (2000–2024), WGI, POLCON III, FSI.

Résultats clés
  • Une meilleure gouvernance (WGI ↑) est corrélée à un volume M&A plus élevé en Amérique Latine (H1 confirmée).
  • La dégradation politique prédit significativement l'abandon de transaction (p < 0.05) — actionnable pour la due diligence (H2 confirmée).
  • Les acquéreurs intègrent le risque politique dans la prime d'acquisition (H3 partiellement confirmée).
  • Implication business : le risque politique est un levier de négociation tarifaire.
